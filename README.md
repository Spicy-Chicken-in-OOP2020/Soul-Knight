## 游戏说明

### 游戏简介
元气骑士是一款地牢类角色扮演游戏，玩家将会在随机关卡中遇到不同的敌人，并且躲避子弹，收集金币，不断获得更强力的武器。
### 游戏玩法
进入游戏后选择要操作的人物角色，在安全地图挑选武器后进入关卡。游戏全程共有五关，走到最后击败最终Boss即可胜利通关。击杀敌人与开启宝箱可以获得武器与金币。不断升级角色，用金币购买更加强力的武器来击败敌人！
### 基本操作

  操作    |  效果   | 
  :-----|:---------------
  WASD或↑↓←→| 移动主角 
  J或Z| 攻击键 
  X或K| 交互键
